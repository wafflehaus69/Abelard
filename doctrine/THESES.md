# THESES.md

Each position Mando holds, targets, or watches as a structured record.
Updated continuously. When a catalyst fires, update the record. When a
thesis-breaker hits, mark the thesis falsified and explain what happened.

Records come in two types:

- **Active theses** — positions built around specific, named catalysts and
  thesis-breakers. These are the pre-narrative or partially-discovered
  opportunities.
- **Macro expressions** — positions that express regime-level views rather
  than specific catalysts. Held for duration, not for event-driven
  re-rating.

Every record is dated. Last-reviewed dates matter — stale records degrade
in usefulness faster than stale news.

## META-NOTES FOR ABELARD

### How to Use This File

Read at startup. When news arrives that affects a thesis, update the
relevant record — not in a separate log, directly in THESES.md. The
records are living documents.

When you update a record:

1. Preserve the prior version in a single-line comment or a short
   history note at the bottom of the record.
2. Update the Last Reviewed date.
3. If a catalyst has fired, move it to a "Catalyst Fired" note and
   identify the next catalyst (if any).
4. If a thesis-breaker has hit, mark the thesis FALSIFIED and explain
   the break. Do not delete the record — a falsified thesis is data.

### What Belongs Here vs. Elsewhere

- Position-specific research depth → research/<ticker>-<topic>.md
- Current-state geopolitical/market context → today's memory file
- Changes to Mando's philosophy or framework → WORLDVIEW.md
- Changes to the research methodology → AGENTS.md

This file is the intersection of those — what Mando holds, targets, or
watches, and why, stated as records that can be updated as the world
changes.

### When a New Position is Added

Create a new record using the template. Fill in every field. A position
with no named catalyst and no named thesis-breaker is not ready to be
in this file — it's a belief, not a thesis.
