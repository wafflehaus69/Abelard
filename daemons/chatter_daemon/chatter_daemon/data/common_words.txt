# UNLESS it is in the config word_ticker_allowlist (real tickers that collide
# with common words, e.g. NOW/META/CORN). Cashtags ($TICKER) bypass this list
# entirely.

# to 5 letters are useful here (bare ticker candidates are capped at 5 chars by
# the tokenizer), so the list is scoped to short, high-frequency words that
# collide with ticker-shaped tokens. Grows after live scrapes. One lowercased
# word per line; blank lines and #comments ignored.
#
# Deliberately EXCLUDES ticker-only strings (soxl, tqqq, avgo, dram, nvda, ...)
# so real symbols are not filtered.
